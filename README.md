# Validify – Certificate Verification System

Validify is a full-stack web application that securely generates and verifies digital certificates using unique Certificate IDs.  
The system allows organizations to upload student data via Excel, generate certificates, and provide public verification.

---

## 🚀 Features:-

- Excel-based bulk upload of student data
- Secure certificate generation & storage
- Unique Certificate ID based verification
- Full certificate preview
- User authentication & role-based access
- Admin dashboard for certificate management

---

## 🛠️ Tech Stack:-

**Frontend:** React.js, Tailwind CSS  
**Backend:** Node.js, Express.js  
**Database:** MongoDB  
**Authentication:** JWT  
**File Handling:** Excel processing, image generation
